# Getting Started

## Installation & Set Up

This is a react application built and managed by Node.JS on your machine.

### Installing Node & Python

#### OS X

```bash
brew install node python
```

### Installing Dependencies

```bash
npm i -g yarn
yarn
```

### Running the dev server

```
yarn dev
```

## Google Maps
